# WhatsApp Web UI Clone (Next.js + Tailwind)

This project is a simple WhatsApp Web interface clone built using Next.js and Tailwind CSS.

The project only focuses on the UI. There is no backend, no real-time messaging, and no authentication system.

## Overview

The goal is to recreate the WhatsApp Web interface as close as possible using modern frontend tools.

It includes the sidebar, chat layout, and message design.

## Features

- WhatsApp Web style layout
- Chat list sidebar
- Chat window UI
- Message bubbles (sender / receiver)
- Responsive design

## Technologies

- Next.js
- Tailwind CSS
